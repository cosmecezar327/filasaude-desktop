Solução 2 — Descobrir quem está usando 8080

No CMD:

netstat -ano | findstr :8080

Vai aparecer algo parecido:

TCP    0.0.0.0:8080     LISTENING     12345

O último número é o PID.

Mate o processo:

taskkill /PID 12345 /F

Depois:

java -jar filasaude-0.0.1-SNAPSHOT.jar
